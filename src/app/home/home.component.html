Welcome to the Nereo Material library demo site. You will find here a demonstration of how the components will look like. To see the code of this demo site and learn how to use the library you can check its github project. If you want to see the code and documentation for the library itself, you cand find it in its repo in github as well.

Available Components

Contextual Toolbar

Contextual toolbar allows you to overlay the main toolbar with one that is context-aware. Just bind it so that it knows how many items are being selected and what actions should be Available for the user.

You can read more about this Material Design component intent in the official documentation.

Banner

The Banner is the proper way to notify the user in a non intrusive way when the message has medium priority only and the user action is optional. Despite that, it requires user dismissal.

You can read more about this Material Design component intent in the official documentation.

Backdrop

The Backdrop is for placing controls that can modify the presented content. It becomes the shell itself, so it won't work inside a Sidenav for instance. You can see the code in Github to learn how to use it.

You can read more about this Material Design component intent in the official documentation.

Speed Dial

When you press a FAB, you can get 3 to 6 related actions in the form of a Speed Dial. By using this component, you can assign the icon and text that each of the mini FABs will have.

You can read more about this Material Design component intent in the official documentation
