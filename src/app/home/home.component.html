Welcome to the Nereo Material library demo site. You will find here a demonstration of how the components will look like. To see the of this demo site and learn how to use the library you can check its github project. If you want to see the library code and documentation, you cand find it in its repo in github as well.

Available Components

Contextual Toolbar

Contextual toolbar allows you to overlay the main toolbar with one that is context-aware. Just bind it so that it knows how many items are being selected and what actions should be Available for the user.

You can read more about this Material Design component intent in the official documentation.
